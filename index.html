git学习

首次将代码提交到git

1、初始化 Git 仓库：在你的项目文件夹中打开命令行，运行以下命令来初始化一个新的 Git 仓库git init


2、添加文件到暂存区：使用以下命令将你的文件添加到 Git 暂存区（Index）中，准备提交git add .


3、提交文件到 Git 仓库：在命令行中，运行以下命令来提交文件到 Git 仓库中git commit -m "提交信息"


4、关联远程仓库（可选）：如果你计划将代码推送到远程仓库（如 GitHub、GitLab 等），可以使用以下命令将本地仓库与远程仓库关联起来git remote add origin <远程仓库URL>


5、将代码推送到远程仓库（可选）：最后，如果你有关联远程仓库，可以使用以下命令将代码推送到远程仓库：git push -u origin master
